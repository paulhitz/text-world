- Text World -

This is based on those old Text Adventure games that were popular many years ago. The idea is that you are given a text description of a location in a fantasy world and you are able to type commands to examine objects, move around the world and talk to game characters. The good ones told an interesting story and were fun to play. This is not a good one! :)

I call this a Text World because it is only semi-interactive. All you can do is move around. For example, type 'south' (or 's' for short) to move south. Other commands like 'examine', 'use', 'talk' etc are not implemented. Type 'help' in the game for more information.

The world is broken into a 5x5 grid. You start in the north-west corner so you can initially go s, se or e. There is a monastery on the western side of the map, a forest in the mid-lands, plains to the east and the south contains a large castle.

Enter the World

Tech: It uses a MySQL database to store location and resource information. In the latest version, the frontend communicates with the backend using Ajax which allows the world to be updated without a full page refresh (a major drawback of the old version). It has been successfully tested with recent versions of Firefox, Internet Explorer and Opera. Sorry Lynx users, this version only works in browsers with JavaScript support.
